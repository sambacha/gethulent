# gethulent

> Minimal JSON RPC mocking/proxy/cli for Ethereum Geth 

## Overview 

```bash
$ gethulent --help
```


## Install
```
go get -v github.com/sambacha/gethulent
```

### Building 


```bash
make gethulent
```

### Run tests

```bash
make test
```
